# Premium Personal Website Template

Production-ready first version of a premium one-page personal website template built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Customize

- Replace placeholder copy, links, and structured content in `src/data/site-content.ts`
- Replace the hero visual placeholder with your own portrait, render, or signature asset in `src/components/site/hero-section.tsx`
- Wire up the contact form when you're ready for a functional version

## Quality checks

```bash
npm run lint
npm run build
```
